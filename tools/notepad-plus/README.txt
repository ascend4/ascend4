Notepad++ is a FOSS text editor for Windows, see
http://notepad-plus.sourceforge.net/

To use this syntax file, start Notepad++ and from the menu select
View->User Defined Dialog. Then click 'import' and then select the XML file
from this same directory. You should see "Import successful". Close the
program, relaunch it, and load an ASCEND .a4c or .a4l file and you should
see ASCEND keywords highlighted in the loaded text.
