This directory contains C code that implements the sun position algorithm
by Roberto Grena from the paper:

Roberto Grena, 2008, 'An algorithm for the computation of the solar position',
Solar Energy (82) pp 462-470. doi: 10.1016/j.solener.2007.10.001.

For more information, see
http://ascend4.org/Calculation_of_sun_position.
